# Web Development | Learning Repository

Collection of HTML, CSS, JavaScript, React, and Tailwind CSS projects built while learning web development from courses and online resources.

## About The Project

This repository documents my web development learning journey with hands-on projects covering HTML fundamentals, CSS styling (vanilla and Tailwind), JavaScript programming, and React component development. Projects progress from simple static pages to interactive applications, demonstrating growth in front-end skills and best practices.

## Built With

- HTML5
- CSS3 (vanilla and Tailwind CSS)
- JavaScript (ES6+)
- React.js
- Vite (for React projects)

## Getting Started

### Prerequisites

- Web browser (Chrome, Firefox, Safari, Edge)
- Code editor (VS Code recommended)
- Node.js 16+ (for React projects only)

### Installation

1. Clone the repo
   ```bash
   git clone https://github.com/tumansutradhar/web-development.git
   cd web-development
   ```

2. For HTML/CSS/JS projects
   - Navigate to the project folder
   - Open `index.html` in a web browser

3. For React projects
   ```bash
   cd "React JS"
   npm install
   npm start
   ```
